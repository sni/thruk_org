Highlights of release {{ page.version }}

Here are some of the highlights of the release {{ page.version }}. See the detailed changelog for a complete list of changes.

Dashboard Maintenance Mode

Dashboards can be put into maintenance mode. That way you can quickly address issues with either the dashboard itself or items shown on the dashboard. This can be handy if this dashboard is shown on many wallscreens.

Changes

  • `rest_api_enabled` option removed. The rest api is always enabled now.
