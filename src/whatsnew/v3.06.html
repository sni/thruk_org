Advanced lexical filtering

Power users wanted to have lexical queries for a long time and now it's finally possible.
The advanced query filter input box can be triggered in two ways:
- either hit the space key in the empty search bar in the top row or the side navigation.
- or hit the advanced filter button in the search filter popup.

Queries have access to all attributes and can make use of complex logical expressions.
The query input has a code completion for attribute names in case you cannot remember all
of them.
