Repository

Using the Consol Labs Repository is probably the easiest way to install or update Thruk.

Packages

There are mk-livestatus packages available in the Consol Labs Repository.
You may need them for Nagios and Icinga. Shinken has livestatus included in
the core so you don't need extra packages besides Thruk. Naemon also comes with its own Livestatus module.

Naemon

Naemon, as the successor of Nagios, comes with Thruk as default gui. It has ready to go packages including Thruk and Livestatus. Take Naemon if you consider upgrading to Nagios 4.

OMD

Thruk is included in the OMD (Open Monitoring Distribution). Installing
Thruk with OMD is an easy and fast way of getting Thruk in a complete setup running.
OMD includes a preconfigured Nagios/Icinga/Shinken + Thruk + PNP4Nagios and other useful addons.
