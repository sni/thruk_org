NAME

Catalyst::Plugin::Thruk::ConfigLoader - rearrange config just before application starts

SYNOPSIS

    use Catalyst qw[Thruk::ConfigLoader];

DESCRIPTION

Used to set and change some config options after the config has been loaded but before the application has started

OVERLOADED METHODS

finalize_config
