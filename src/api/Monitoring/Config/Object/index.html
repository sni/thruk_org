NAME

Monitoring::Conf::Object - Object Prototype

DESCRIPTION

Single object creator

METHODS

new

    new({
        coretype => coretype,  # can be nagios, icinga, shinken or any
        type     => type,
        line     => nr,
        file     => file object,
        name     => name
    })

return a new object of given type. Type can be any of the following: host hostgroup hostextinfo hostdependency hostescalation service servicegroup serviceextinfo servicedependency serviceescalation command timeperiod contact contactgroup module escalation
