NAME

Thruk::Request - Subclass of Plack::Request which supports encoding.

DESCRIPTION

based on Plack::Request::WithEncoding.

WMETHODS

encoding

Returns a encoding method to use to decode parameters.

query_parameters

Returns a reference to a hash containing decoded query string (GET) parameters.

body_parameters

Returns a reference to a hash containing decoded posted parameters in the request body (POST). As with query_parameters.

parameters

Returns a hash reference containing decoded (and merged) GET and POST parameters.

param

Returns decoded GET and POST parameters.

url

Returns urldecoded utf8 uri as string.

uri

Returns Uri object, but applies HTTP_X_FORWARDED_* environment if set.

unescape

Returns uri_unescape for given value.

as_string

Returns request as string for debugging purposes.

SEE ALSO

Plack::Request, Plack::Request::WithEncoding
