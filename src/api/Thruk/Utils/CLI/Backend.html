NAME

Thruk::Utils::CLI::Backend - Backend CLI module

DESCRIPTION

The backend command lists livestatus backends

SYNOPSIS

  Usage: thruk [globaloptions] backend <cmd>

OPTIONS

help
    print help and exit
cmd
    available commands are:

    - list          lists all backends

METHODS

cmd

    cmd([ $options ])

EXAMPLES

Display all available backends

  %> thruk backend list
